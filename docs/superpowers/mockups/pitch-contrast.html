Contrast math: #F0F0F5 (textPrimary) on #1A1A2E (bgCard) ≈ 15.5 : 1 — well above WCAG 2.1 AA's 4.5:1 threshold for normal text.
#F4A261 (accent overline + drop hook) on #1A1A2E ≈ 7.9 : 1 — passes AA for graphical UI (3:1) and even AA for text (4.5:1).
Legend: Overline on a mora = high pitch for that mora. Small amber hook at the trailing edge of a high mora = pitch drop to the next mora (e.g. atamadaka: 息子 むすこ pattern [1,0,0] → high on む, drop after, low on すこ).

Note: this HTML is an approximation — the real React Native component uses borderTopWidth: 2 and an absolutely-positioned drop <View>. Spacing on iOS will be slightly tighter than what you see here due to native font metrics, but contrast behaviour is identical (same hex values).
